# Sssnake game
Author: Pekka Alho

Version: 0.1

Pygame project based on thenewboston's [Pygame (Python Game Development) Tutorial](https://www.youtube.com/watch?v=K5F-aGDIYaM)

##Instructions
######Run
```
main.exe
```
######Building sssnake
Note: requires cx_Freeze
```
python setup.py build 
```
